Port EverlastingSkins/SkinRestorer

1.12.2 Cleanroom (forge) Java 25

Этот мод был создан от моего желания и необходимости скинчиков). 

Я портировал мод everlastingskins так как его код непозволял мне зайти на сервер из за молейшей ошибки и там было неоптимизировано и из за этого сервер нагружался и мне  пришлось восоздать новый порт который ненагружает сервер (0% нагрузки во время работы) и прекрасно функционировал

Я брал логику работы кода из модов EverlastingSkins и SkinRestorer

Этот порт строго требует:
- Cleanroom - Это переделаное ядро forge 1.12.2
- Java 25 - советую graalvm

Полность серверный

Весит очень мало всего 15,5 кб

Команды работы этого мода:
- /skin <Имя профиля Mojang> - накладывает на вас скин лицензированого профиля
- /skin <какому игроку> <Имя профиля Mojang> - администратор или сервер может на вас наложить скин лицензированого профиля

Примечание:

Возможно код несовместим с java 8 так как я писал на java 25 но вы можете проверить
